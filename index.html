Calculador de distancias

Sirve para calcular las distancias en Kilómetros entre varios lugares del mundo sólo utilizando coordenadas, es un software sencillo que ahorra tiempo al necesitar determinar distancias para obtener lás más cercanas a un destino. Compara cada dato de una lista de partida con una de llegada y se almacena generando un resumen en Excel y guardando los datos en una base de datos Oracle para poder utilizarlos nuevamente sin necesidad de buscar las coordenadas
